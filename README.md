# Vidi — сайт (продакшн-версия)

Статический лендинг. Без сборки и бэкенда — папку можно загружать на любой хостинг как есть (Timeweb, GitHub Pages, Cloudflare Pages).

## Файлы
- `index.html` — страница (hero-вариант A, панель Tweaks удалена)
- `styles.css` — стили (палитра в `:root`)
- `landing.js` — шапка при скролле, FAQ-аккордеон, scroll-reveal, мобильное меню
- `assets/` — иконка, скриншоты (WebP, оптимизированы), og-image

Тяжёлые PNG-оригиналы скриншотов остались в `assets/` рядом с WebP — на хостинг их заливать не нужно (страница ссылается только на .webp, .png можно удалить).

1. **Домен**: после покупки заменить в `index.html` `og:image` на абсолютный URL (см. TODO-комментарий в `<head>`) и добавить `robots.txt` + `sitemap.xml` с доменом.
2. **Аналитика**: счётчик не подключён. Кнопки уже размечены `data-evt` (download / download_nav / subscribe) — повесить обработчики при подключении Я.Метрики.
3. **404.html** — по желанию, для красоты.
4. Проверить, что бакет Yandex Cloud публичный и «DICOM Viewer 0.7.dmg» скачивается по ссылке из кнопки.
5. При выпуске новой версии приложения: обновить имя .dmg в трёх ссылках в `index.html` (или загружать в бакет файл с постоянным именем — тогда сайт менять не придётся).
